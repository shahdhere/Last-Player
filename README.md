# 🎮 Last Player
A Unity-based survival game inspired by Squid Game (Red Light Green Light).

---

## Game Concept
The player must survive a deadly game controlled by a giant doll.
- The doll sings while facing away → player can move freely  
- When the doll stops singing and looks at the player → the player must stop immediately  
- If the player moves during detection → the doll shoots a laser and eliminates the player  

---

## Game Objective
Reach the red line before the timer runs out.
- Total time: 2 minutes and 15 seconds
- If the player fails to reach the finish line in time → the doll eliminates the player

---

## Features
- AI Doll behavior (look detection system)
- Laser attack system
- Countdown timer
- Win system
- Lose system
- Pause menu system
- Restart & Main Menu buttons
- UI system (Win / Lose / Pause screens)

---

## Built With :
- Unity Engine  
- C#  
- Unity UI System  

---

## How to Play :
1. Run the game  
2. Move only when the doll is not watching  
3. Stop immediately when she looks at you  
4. Reach the red line before time runs out  
5. Survive to win
